# LMS Research Data Collection Tool

A simple quantitative data collection tool for researching Learning Management Systems (LMS) UX in Computer Science education.

## Features
-  Consent management with ethical considerations
-  Performance metrics collection (response time, page load, errors)
-  User satisfaction ratings (Likert scales)
-  Demographic information collection
-  Data validation and real-time preview
-  CSV export functionality
-  Local storage for testing
-  Simple Flask backend API
